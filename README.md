# Netflix-EDA
Exploratory Data Analysis (EDA) of the Netflix Movies and TV Shows dataset using Python, Pandas, Matplotlib, and Seaborn.

## Project Overview
This project explores the Netflix Movies and TV Shows dataset using Exploratory Data Analysis (EDA). It includes data cleaning, feature engineering, and visualizations to uncover patterns and insights about Netflix's content.

## Dataset
- Source: Kaggle
- Dataset: Netflix Movies and TV Shows

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
- Data Loading
- Data Cleaning
- Handling Missing Values
- Feature Engineering
- Exploratory Data Analysis
- Data Visualization
- Key Insights

## Key Insights
- Distribution of Movies and TV Shows
- Top content-producing countries
- Most common genres
- Content added over the years
- Rating distribution
- Movie duration analysis
